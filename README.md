# Construction Resource Management System (CLI)

## Overview
A Python-based Command Line Interface (CLI) tool designed for XYZ Construction Company. This system helps project managers track ongoing projects, manage project lifecycles (Ongoing, On-Hold, Completed), and effectively allocate the workforce of 500 available workers.

## Features
- **Project Creation:** Add new projects with validation checks for worker availability.
- **Resource Management:** Automatically deducts workers when a project starts and releases them back to the pool when a project completes.
- **Status Tracking:** Update projects to "On Hold" or "Completed" and move them to respective logs.
- **Statistics Dashboard:** View real-time counts of active projects and available workforce.
